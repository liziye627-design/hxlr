# Sound Effects Placeholder

This directory contains sound effect files for the Werewolf game.

## Current Status

The following placeholder files need to be replaced with actual sound effects:

- `night-transition.mp3` - Played when entering night phase ohn
- `day-transition.mp3` - Played when entering day phase
- `player-death.mp3` - Played when a player dies
- `voting.mp3` - Played during voting phase
- `skill-use.mp3` - Played when a player uses their role skill
- `game-start.mp3` - Played when the game begins
- `game-end.mp3` - Played when the game ends
- `sheriff-elected.mp3` - Played when sheriff is elected
- `timer-tick.mp3` - Played during countdown timers

## How to Replace

1. Download or create your sound effects
2. Name them according to the list above
3. Place them in this `public/sounds/` directory
4. Recommended format: MP3, 128kbps, stereo
5. Keep file sizes small (< 500KB each) for fast loading
